# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-25
### Added
- Initial release of Nexora Section Builder.
- Core Sections: Hero, Banner, Featured Products, Category Grid, Custom HTML.
- Admin Page Builder UI with drag-and-drop.
- Multi-channel and multi-locale caching system.
- Official Bagisto Repository and Proxy patterns.
- `php artisan nexora:install` command.
- Tailwind and Vite-based asset compilation.
- Root `.gitignore` and comprehensive `README.md`.
